# Dispatch format

Treat the user's input as the scene description. Pick the event tag
that fits best from the canonical set:

- `brawl` — physical conflict, named participants
- `arrival` — someone enters the scene
- `trade` — exchange of goods, named parties
- `theft` — taking without consent
- `rumor` — heard, not directly observed
- `departure` — someone leaves
- `sighting` — observed without interaction
- `pact` — an agreement struck between actors
- `failure` — an attempted action that did not complete

Use real nouns (names, objects, locations). Single sentence, present
tense, 12–24 words. The game can't rewrite what you saw — your
signature is the receipt. Witness, don't narrate.
